Hydrogen-Like Atomic System#

This exercise will use Mathematica. Download the program from Student Associated Files.
Hydrogen-Like atom is a system containing only one electron orbiting around nucleus. Therefore, this system is solvable and the exact solution to this problem is given by the product of radial wave function and spherical harmonics we studied so far. Because Mathematica can convert spherical polar coordinate to Cartesian coordinate, we can repeat the same problem using simple Cartesian coordinate system. Understanding how the actual code for this coordinate transformation works is out of scope for this class. Therefore, the automated radial wave function and spherical harmonics are encoded and attached to your Mathematica file. Because swapping coordinate system does not affect the outcome, we should obtain the same solutions, and their eigenvalues obtained by using spherical polar coordinate system. To complete this exercise, you will need to check orthogonality of the solutions. (Pick 5 random cases) Check all the orbitals for normalization. Also, obtain the average distance away from the center; for example, Bohr radius for 1s orbital of the Hydrogen atom. Compute the energy of all the orbitals. To do this in Cartesian coordinate, you need to compute
$\(\begin{equation*} \begin{split} & E =\int\int \int \psi^{*}(x,y,z) \hat {H}\psi(x,y,z) dxdydz \\ & =\int\int \int \psi^{*}(x,y,z) \left( \frac{-1}{2} \nabla^2- \frac{1}{\sqrt{x^2+y^2+z^2}} \right)\psi(x,y,z) dxdydz \\ \end{split} \end{equation*}\)$ Do this for all orbitals up to 4f. \(H_2^{+}\) System#

This exercise will use Mathematica. Download the program from Student Associated Files.
\(H_2^{+}\) is a system containing only one electron orbiting around two nuclei (one electron non-central body problem). Assume wave function is a simple combination of 1S orbitals from both H atoms. Because we only deal with linear operators, we can describe this system by just simply computing Coulomb and Exchange integral at given nuclear separation in addition to isolated H atom. The definitions of Exchange (K) and Coulomb (J) integral in this problem are given as follows. If only one orbital is involved in the integrand, then it is Coulomb, and Exchange otherwise. Vary the atomic separation from 1 to 4, plot out the contributions of J, K and total energy for this system. Calculate J & K to estimate the energy of the system with \(\psi_+\) and \(\psi_-\). Plot the energy diagram and predict the bond length.

Term Project: Group Theory on Electronic Transition#

If you do not understand any word, look for it online (AI will do the job). In many field of chemistry, you will hear \(\pi\) to \(\pi^{*}\) transition as well as \(n\) to \(\pi^{*}\) transition (lone pair to exited MO of particular kind). In this exercise, you will learn these important transitions from physical chemistry point of view. So the tool we will use for this project is a group theory. You might want to review the lecture notes or read the relevant section of the textbook. In this project, you will study formaldehyde (\(CH_2O\)) which includes \(sp^{2}\) hybridized orbitals on C and O atoms. You can symmetrize these hybrid atomic orbitals centered at C and O by the procedures given in the textbook.\
