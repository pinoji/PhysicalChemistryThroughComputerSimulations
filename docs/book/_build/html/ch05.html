Approximation Methods#

We now consider system containing more than two electrons. Because electrons are charged, they repel each other. Repulsion between nuclei is easy to compute because they are represented by a point charge. However, the electron density is a distribution determined by \(\left| \psi \right|^2\). Calculating repulsive interaction between two orbitals is performed by numerical integrations which are out of scope for this textbook. Also, repulsive interactions introduce the additional term in Hamiltonian; therefore, hydrogen-like atomic orbitals is not an exact solution to the Schrödinger equation anymore. In other words, \(\psi\) becomes the unknown and central theme of quantum mechanics become how to obtain an accurate representation of \(\psi\) in terms of the available basis. In this chapter, we introduce the fundamental mathematical concept that is vital for obtaining the solution to the Schrödinger equation.\

Variational Principle#

Ground state \(\psi_0\) has the lowest energy denoted as \(E_0\) by definition

\[\begin{split}\begin{equation*} \begin{split} \hat{H}\psi_0 &= E_0\psi_0\\ \int\psi_0^\ast\hat{H}\psi_0dx &= E_0\int\psi_0^\ast\psi_0dx\\ \frac{\int\psi_0^\ast\hat{H}\psi_0dx}{\int\psi_0^\ast\psi dx} &= E_0 \end{split} \end{equation*}\end{split}\]

by def. any \(\psi\) other than \(\psi_0\) for non degenerate system
$\(\begin{equation*} \frac{\int\psi^\ast\hat{H}\psi dx}{\int\psi^\ast\psi dx} = E > E_0 \end{equation*}\)$

This is called “variational principle”.
This is super useful for chemistry because we do not know how to solve Shrödinger Equation for any system having more than 1 electron.
[i.e.]{.underline} He atom must be solved numerically using variational theorem.

Check Variational Principle#

Hydrogen Atom by Gaussian Orbitals with Variable Standard Deviation#

\[\begin{equation*} \hat{H} = -\frac{\hbar^2}{2m_er^2}\frac{d}{dr}\Big(r^2\frac{d}{dr}\Big) - \frac{e^2}{4\pi\epsilon_0r} \end{equation*}\]

Let us use \(\phi(r) = e^{-\alpha r^2}\).
We optimize \(\alpha\) later.

[Calc II]{.underline} \(\downarrow\)

\[\begin{split}\begin{equation*} \begin{split} 4\pi\int\phi^\ast\hat{H}\phi r^2dr & = \frac{3\hbar^2\pi^{3/2}}{4\sqrt{2}m_e\alpha^{1/2}} - \frac{e^2}{4\epsilon_0\alpha}\\ 4\pi\int\phi^\ast\phi r^2dr& = \Big(\frac{\pi}{2\alpha}\Big)^{3/2} \end{split} \end{equation*}\end{split}\]

\(E(\alpha) = \frac{3\hbar^2\alpha}{2m_e} - \frac{e^2\alpha^{1/2}}{2^{1/2}\epsilon_0\pi^{3/2}}\)
Shoot for lowest \(E\) by optimizing \(\alpha\).

\[\begin{split}\begin{equation*} \begin{split} \frac{dE}{d\alpha} &= 0 = \frac{3\hbar^2}{2m_e} - \frac{e^2}{(2\pi)^{3/2}\epsilon_0\alpha^{1/2}} \Rightarrow \alpha = \frac{m_e^2 e^4}{18\pi^3\epsilon_0^2\hbar^4}\\ E_{min} &= -\underbrace{\frac{4}{3\pi}}\Big(\frac{m_e e^4}{16\pi^2\epsilon_0^2\hbar^2}\Big) > -\frac{1}{2}\Big(\frac{m_e e^4}{16\pi^2\epsilon^2\hbar^2}\Big) = E_0 \hspace{10pt} \text{//} \end{split} \end{equation*}\end{split}\]

-0.424 \(>\) -0.5 (known G.S. of the system)

Helium Atom (2 \(e^-\) System) with ignoring \(e^--e^-\) repulsive interaction#

\(\downarrow\) nuc charge

\[\begin{equation*} \hat{H} = -\frac{\hbar^2}{2m_e}\Big(\nabla_1^2 + \nabla_2^2\Big) - \frac{Ze^2}{4\pi\epsilon_0}\Big(\frac{1}{r_1} + \frac{1}{r_2}\Big) + \frac{e^2}{4\pi\epsilon_0}\frac{1}{r_{12}} \end{equation*}\]

Simplest way to obtain approximate solutionis to ignore repulsion

\[\begin{equation*} \hat{H}_{simple} = \hat{H}(1) + \hat{H}(2) + \cancel{\frac{e^2}{4\pi\epsilon_0}\frac{1}{r_{12}}} \end{equation*}\]

Because each part of Hamiltonian only depends on single electron coordinates, simple product of hydrogen-like atomic solution is an eigen function to this hamiltonian. i.e. \(\phi_0(r_1, r_2) = \psi_{1s}(r_1)\psi_{1s}(r_2)\)
where \(\psi_{1s}(r) = \Big(\frac{Z^3}{\pi a_0^3}\Big)^{1/2}e^{-\frac{Zr}{a_0}}\), \(a_0 = \frac{4\pi\epsilon_0\hbar^2}{m_e e^2}\)

These simple product wavefunction is often called Hartree product wavefunction.
[HW]{.underline} Check \(\phi_0(r_1, r_2)\) is an eigenfunction of \(\hat{H}_{simple}\) given above.
With the help of computer program, one can obtain

\[\begin{equation*} E = \int\int\phi_0^\ast\big(r_1, r_2\big)\hat{H}\phi_0\big(r_1, r_2\big) dr_1dr_2 = -2.75 a.u. \end{equation*}\]

Note that Hamiltonian is not the simple Hamiltonian. So the solution is not the optimum solution for normal Hamiltonian. This energy is higher than the eigenvalue of exact wavefunction (-2.90 a.u.). So, variational theorem is not violated.\

Secular Equation#

So far, we have verified our trial wavefunction (simple hartree product) is an eigenfunction and it did not violate the variational principle. The Hartree product may be equivalent of the electron configuration we studied in general chemistry.[1] For example, He atom electron configuration is \(1s\alpha(1)1s\beta(2)\) following aufbau principle. Because single hydrogen-like atomic orbital is not a good representation of atomic orbital for multi-electron atomic system, we may represent orbital by linear combination of hydrogen-like atomic orbitals.
This time, we make \(\psi\) more flexible.
\(\phi = \sum_{n=1}^N C_n f_n \hspace{10pt} \leftarrow\) linear combination of some functions(i.e. hydrogen-like atomic orbitals)

\(\uparrow\)

our guess \(\psi\)

\(\phi = C_1f_1 + C_2f_2\)
$\(\begin{equation*} \begin{split} \int\phi^\ast\hat{H}\phi d\tau &= \int\Big(C_1f_1 + C_2f_2\Big)^\ast\hat{H}\Big(C_1f_1 + C_2f_2\Big)d\tau\\ &= C_1^2\int f_1^\ast\hat{H}f_1d\tau + C_1C_2\int f_1^\ast\hat{H}f_2d\tau + C_2C_1\int f_2^\ast\hat{H}f_1d\tau + C_2^2\int f_2^\ast\hat{H}f_2d\tau\\ &= C_1^2H_{11} + 2C_1C_2H_{12} + C_2^2H_{22}\\ \int\phi^\ast\hat{S}\phi d\tau &= C_1^2S_{11} + 2C_1C_2S_{12} + C_2^2S_{22} \end{split} \end{equation*}\)$
