$\star$ Frank Condon Principle\
Because $\psi_{\text{post(not allowed)}}$ has drastically different
nuclear coordinate from the one for the ground state,their associated
wave function $\lvert \psi_{pre} \rvert\ $$^2$ &
$\lvert \psi_{not- allowed} \rvert\ $$^2$ do not have significant
overlap (one of the function is close to zero). Such transition is
rarely seen in the experiment.

$\star$ Ro-vibrational states\
Using photon in IR range, it is possible to excite both vib & rot modes.
Most people use units called wave #, we will adopt the units.\
$E_\nu = \left(\nu + \frac{1}{2}\right)\hbar \omega = \left( \nu + \frac{1}{2}\right)h\nu \hspace{10pt} \nu=0, 1, 2,\cdots$\
In wave \# unit,\
$$\begin{equation*}
\begin{split}
E_{\text{vib}} \text{ in wave \# } := G = \frac{E_\nu}{hc} &= \left(\nu + \frac{1}{2}\right)\frac{\nu}{c}\\
& = \left(\nu + \frac{1}{2}\right)\tilde{\nu}
\end{split}
\end{equation*}$$

Similarly,

$$\begin{equation*}
E_j = \frac{\hbar^2}{2I}J(J + 1) \hspace{10pt} J = 1, 2, \cdots
\end{equation*}$$

In wave \# unit,

$$\begin{equation*}
\begin{split}
E_{\text{rot}} \text{ in wave \# } : = F = \frac{E_J}{hc} &= \left(\frac{h}{8\pi^2cI}\right)J(J + 1)\\
&= \tilde{B}J(J + 1) 
\end{split}
\end{equation*}$$

$$\begin{equation*}
\tilde{E}_{\nu, J} = G(\nu) + F(\nu) = \left(\nu + \frac{1}{2}\right)\tilde{\nu} + \tilde{B}J(J + 1)
\end{equation*}$$

Consider\
$\Delta v = 1$ & $\Delta J = +1$.

$$\begin{equation*}
\begin{split}
\tilde{v}_{\text{obs}}\left(\Delta J = \pm 1\right) &= E_{v+1, J+1} - E_{v,J}\\
&= \left(\nu +\frac{3}{2}\right)\tilde{v} + \tilde{B}(J + 1)(J + 2)\\
&- \left[\left( v + \frac{1}{2}\right)\tilde{v} + \tilde{B}J(J + 1)\right]\\
&= \tilde{v} + 2\tilde{B}(J + 1) \hspace{10pt} J = 0, 1, 2, \cdots
\end{split}
\end{equation*}$$

Because $J$ starts from several values at room T,\
$\tilde{v}_{\text{obs}}$ contain few peaks.

Similarly,\
Consider $\Delta v = 1$ & $\Delta J = -1$\
$$\begin{equation*}
\begin{split}
\tilde{v}_{\text{obs}}\left(\Delta J = -1\right) &= E_{v+1,J-1} - E_{v, J}\\
&=\left(v + \frac{3}{2}\right)\tilde{v} + \tilde{B}(J - 1)J\\
&-\left[\left(v + \frac{1}{2}\right)\tilde{v} + \tilde{B}J(J + 1)\right]\\
&= \tilde{v} - 2\tilde{B}J \hspace{10pt} J = 1, 2, 3, \cdots
\end{split}
\end{equation*}$$

$\uparrow$

$J$ can't end up negative.\
$\tilde{v}_{\text{obs}}(\Delta J = 1)$ forms $R-$branch.\
$\tilde{v}_{\text{obs}}(\Delta J = -1)$ forms $P-$branch.\
