###  Resonance by Variational Methods

####  Variational method revisited

By the variational principle,\
$$W = \frac{\int\psi^\ast\hat{H}\psi dr}{\int\psi^\ast\psi dr} \ge E_0$$

Express $\psi$ as a linear combination of atomic orbitals (LCAO).

$$\psi = \sum_{i=1}^nc_if_i \hspace{10pt} \text{(real atomic orbitals used.} f^\ast = f)$$

$$\begin{equation*}
\begin{split}
\int\psi^\ast\hat{H}\psi dr & = \int\left(\sum c_if_i\right)\hat{H}\left(\sum c_jf_j\right)dr\\
&= \sum_{i,j}c_ic_j\underbrace{\int f_i^\ast\hat{H}f_j dr}_{\text{denote this as } H_{ij}}\\
&= \sum_{i,j}c_ic_jH_{ij}
\end{split}
\end{equation*}$$

$$\begin{equation*}
\begin{split}
\int\psi^\ast \psi dr &= \int\left(\sum_i c_if_i\right)\left(\sum_j c_jf_j\right)dr\\
&= \sum_{i,j} c_ic_j\underbrace{\int f_if_j dr}_{\text{call this} S_{ij}}\\
&= \sum_{i,j} c_i c_j S_{ij}
\end{split}
\end{equation*}$$

<hr/>
                         We will minimize $W = \frac{\sum_{i,j}c_ic_j H_{ij}}{\sum_{i,j}c_ic_j S_{ij}}$
                             $\Downarrow$ 
    $\frac{\partial W}{\partial c_k} = 0$ for all $k = 1, 2, \cdots, n$.
<hr/>

$$W\left[\sum_{i,j}c_i c_j S_{ij}\right] = \left\{\sum{i,j}c_i c_j H_{ij}\right\}$$

$$\frac{\partial}{\partial c_k}\left[\sum_{i,j}c_i c_jS_{ij}\right] = \frac{\partial}{\partial c_i}\left\{\sum_{i,j} c_i c_jH_{ij}\right\}$$

$$\left(\frac{\partial}{\partial c_k}W\right)\big[ \hspace{10pt} \big] + W\sum_{i,j}\left(\frac{\partial c_i}{\partial c_k}\right)c_j S_{ij} + W\sum_{i,j}c_i\left(\frac{\partial c_j}{\partial c_k}\right)S_{ij}$$

$$=  \sum_{i,j}\left(\frac{\partial c_i}{\partial c_k}\right)c_j H_{ij} + \sum_{i,j}c_i\left(\frac{\partial c_j}{\partial c_k}\right)H_{ij}$$

[Note]{.underline}

  at min $\frac{\partial W}{\partial c_k} = 0$                                                                                                               $\rotatebox{90}{=}$

$$\begin{equation*}
\underbrace{W\sum_j c_j S_{kj}}_{\text{separate }\uparrow \text{ terms }\nearrow} + \underbrace{W\sum_i c_i S_{ik}} = \sum_j c_j H_{kj} + \sum_i c_i H_{ik}
\end{equation*}$$

I could write

$$\begin{equation*}
W\sum_j c_j S_{kj} + W\sum_j c_j S_{jk} = \sum_j c_j H_{kj} + \sum_j c_j H_{jk}
\end{equation*}$$

Note also

$$S_{kj} = S_{kj} \text{\&} H_{kj} = H_{jk}$$

$$\begin{equation*}
\begin{split}
\cancel{2}W\sum_j c_j S_{kj} = \cancel{2}\sum_j c_j H_{kj} \\
\sum_j c_j H_{kj} - W c_j S_{kj} = 0\\
\sum_{j=1}^{n}\left( H_{kj} - WS_{kj}\right) c_j = 0
\end{split}
\end{equation*}$$

This is actually $n$ equation because we could have taken
$\frac{\partial}{\partial c_m}$ $m \neq k$. instead of
$\frac{\partial}{\partial c_k}$.

$$\begin{array}{ccccccccc}

\end{array}$$

$$\begin{array}{ll}

It is not too difficult to obtain orthogonal vectors by increasing the
number of nodes in molecular orbitals for electron associated to the
conjugated system. The general procedure starts from a molecular orbital
with zero nodes and gradually increases the number of nodes. This
procedure is relatively easy (just use 1 $\&$ -1) for conjugated bonding
system having $4n$ carbon atoms. For any other conjugated bonding
system, use 0, 1, and -1 to make the required nodes. Then, scale some of
the $p_z$ orbitals to make them orthogonal. In more systematic approach,
Gram-Schmidt orthogonalization can be used. However, the most efficient
method of obtaining the proper molecular orbitals is to compute directly
via eigenvalue algorithm. The interested reader may refer to standard
numerical analysis textbook.\
