# System Containing Two Atoms (Diatomic System)

We now consider the simplest system that containing more than one
nuclei. $H_2^+$ is the simplest system of the diatomic system because
there is no electron-electron repulsion. Because the electron is equally
distributed, molecular orbital for this atom must be composed of atomic
orbitals centered at each of these hydrogen atoms. Unlike single atomic
system, atomic orbitals centered at different atoms are not orthogonal.
In this chapter, we introduce overlap integral, Coulomb integral, and
exchange integral. All of these are fundamental for understanding
chemical bonds. Also, term symbols are introduced for diatomic system.

Let $C_1 = C_2 = 1$ for now. ( normalization is automatically

$\hat{H}\psi_+(r) = E_+\psi_+(r)$\
$E_+ = \frac{\int \psi^\ast\hat{H}\psi dr_1}{\int\psi^\ast\psi dr_1} \rightarrow dr_1$
means coordinate associated with $e^-$ #1.

Let $dr_1 = dr$ Lasy is good. Let's compute the denominator.

$$\begin{array}{ll}
\int\psi^\ast\psi dr &= \int(1S_A + 1S_B)^\ast(1S_A + 1S_B) dr\\
&= \underbrace{\int 1S_A^\ast1S_A dr}_{=1} + 2\underbrace{\int1S_A1S_B dr}_{S(R)} + \underbrace{\int1S_B1S_B dr}_{=1}\\
& = 2 + 2S(R) \cdots \text{\textcircled{1}}
\end{array}$$

Notice $S(R)$ becomes small as $R$ gets large.

[i.e.]{.underline}\
$S(R) = e^{-R(1+R+\frac{R^2}{3})}$

$\uparrow$

fitting data

$$\begin{equation*}
\begin{split}
\int\psi^\ast\hat{H}\psi dr &= \int\left(1S_A + 1S_B\right)^\ast\left[-\frac{1}{2}\nabla^2 - \frac{1}{r_{1A}} - \frac{1}{r_{1B}}\right]\left(1S_A + 1S_B\right) dr\\
&= \int\left(1S_A + 1S_B\right)^\ast\Big[ \hspace{90pt}\Big]1S_A dr \\
& + \int\big(\hspace{50pt}\big)^\ast\Big[\hspace{90pt}\Big]1S_B dr
\end{split}
\end{equation*}$$

Note

$$\begin{equation*}
\begin{split}
\left[-\frac{1}{2}\nabla^2 - \frac{1}{r_{1A}}\right]1S_A &= E_{1S_A}1S_A \hspace{10pt}\text{same for $E_{1S_B}1S_B$ \& $E_{1S_A} = E_{1S_B}=E_{1S}$}\\
&= \int\left(1S_A + 1S_B\right)^\ast\left[E_{1S} - \frac{1}{r_B}\right]1S_A dr\\
&+ \int\big(\hspace{48pt}\big)^\ast\left[E_{1S} - \frac{1}{r_A}\right]1S_B dr\\
&= \int 1S_A\left[E_{1S} - \frac{1}{r_B}\right]1S_A dr + \int 1S_B\left[E_{1S} - \frac{1}{r_B}\right]1S_A dr\\
& + \int 1S_A\left[E_{1S} - \frac{1}{r_A}\right]1S_B dr + \int 1S_B\left[E_{1S} - \frac{1}{r_A}\right]1S_B dr\\
&=E_{1S}\int 1S_A1S_A dr - \int 1S_A\frac{1}{r_B}1S_A dr + E_{1S}\int 1S_B1S_A dr - \int 1S_B\frac{1}{r_B}1S_A dr + \text{e.t.c.}
\end{split}
\end{equation*}$$

$\cdots$ same stuff for $1S_B$

$$\begin{equation*}
\begin{split}
&= E_{1S} + \int \underline{\underline{1S_A}}\frac{-1}{r_B}\underline{\underline{1S_A}} dr \rightarrow \text{same orbital call this Coulomb (J)}\\
&+ E_{1S}S(R) + \int\underline{1S_B}\frac{-1}{r_B}\underline{\underline{1S_A}} dr \rightarrow \text{different orbital call this exchange (K)}\\
&+ E_{1S} + \int 1S_A\frac{-1}{r_A}1S_B dr\\
&+ E_{1S}S(R) +\int1S_B\frac{-1}{r_A}1S_Bdr\\
&= E_{1S}\left(2 + 2S(R)\right) + 2J + 2K \cdots \text{\textcircled{2}}
\end{split}
\end{equation*}$$

$\xRightarrow{\text{\textcircled{1} \& \textcircled{2}}}$ So, Energya of
$H_2^+$ for $\psi_+$ is

$$\begin{equation*}
\begin{split}
E_+ = \frac{\int\psi_+^\ast H \psi_+ dr}{\int\psi_+^\ast\psi_+dr} &= \frac{2E_{1S}\left(1 + S(R)\right) + 2J + 2K}{2 + 2S(R)}\\
&=E_{1S} + \frac{J + K}{1 + S(R)}
\end{split}
\end{equation*}$$

relative $E_+$ compares to $E_{1S}$

$$\begin{equation*}
\Delta E = E_+ - E_{1S} = \frac{J + K}{1 + S(R)}
\end{equation*}$$

We can find the same result if we use secular determinant.\
$$\begin{vmatrix} H_{AA} - ES_{AA} & H_{AB} - ES_{AB}\\ H_{BA} - ES_{BA} & H_{BB} - ES_{BB}\end{vmatrix} = 0$$
