# Hybrid Orbitals in Polyatomic Molecule

So far, we have been using a linear combination of atomic orbitals,
$\phi$ to construct molecular orbitals $\psi$.

<hr/>
  $\psi = \sum_{i=1}^NC_i\phi_i$                
                                   $\nwarrow$   
                                                Hydrogen like atomic orbitals centered at some atom in a molecule
<hr/>

In molecular orbital theory, $\psi$ is delocalized (distributed over all
molecule)

$\Downarrow$

Hard to keep track\
Hybridization is a combenient simplification of molecular orbitals such
that each molecular orbitals are bound tightly to an atom & atoms neaby.
This can be achieved by\

<hr/>
  1⃝   formation of hybrid orbital by linear combination of atomic orbitals from a single atom
  2⃝   approximation of molecular orbitals by a simple combination of a pair of hybrid orbitals
      \[Easier to interprit\]
<hr/>
