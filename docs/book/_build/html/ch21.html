Thermodynamics of Mixing#

Chemical Potential for Ideal Gas#

Obtain…

\(\star\) \(\mu\) for pure ideal gas (mono atomic).

\(\mu = \parfracii{A}{n}{T}{V} = -k_BT\parfracii{\ln z}{n}{V}{T}\)

\(Z = \frac{q^n(V,T)}{n!}\)

\(\ln Z = n\ln q -(N\ln N - N)\)

\(\frac{\partial \ln Z}{\partial n} = \ln q - \ln N - 1 + 1 = \ln \frac{q}{N}\)

\[\begin{split}\begin{align*} \mu &= -k_BT\ln \frac{q}{N}\\ &= -k_BT\ln \left(\underset{\uparrow}{\underbrace{\frac{q}{V}}}\frac{V}{N}\right) = -k_BT\ln\left(\frac{q}{V}\frac{k_BT}{P}\right)\\ &\text{\hspace{20pt}} \left(\frac{2\pi mk_BT}{h^2}\right)^{3/2} \cancel{V}/\cancel{V}\\ %&\frac{k_BT}{\underset{1 \text{barr}}{\underset{\downarrow}{P_0}}} = \underset{\searrow}{k_BT} \text{\hspace{20pt}} \nwarrow \text{ pure func of } T\\ &= -k_BT \ln \left(\frac{q}{V}k_BT\right) + k_BT\ln P\\ & \text{first term is a pure function of T}\\ &= \mu_0 + k_BT\ln P // \end{align*}$$ $\star$ In case you are wondering what happens to the unit inside of $ln$? Pressure is an energy per volume and it is unitless in atomic unit. Now, we may want to make $\mu_o$ more useful by setting, for example,$P=1$ATM.(This is same as shifting zero) In this case, pressure unit conversion ($\times$ conversion factor inside $\ln$ is included in $\mu_o$) Conveniently, this makes\ $$\begin{align*} \mu&=\mu_o + k_BT\ln P \cdots \text{P in ATM}\\ & = \mu_o +0 - 0+ k_BT\ln P \\ & \text{with } P_o=1\\ & = \mu_o +\cancel{ k_BT\ln P_o} + k_BT\ln {\left( \frac {P}{P_0} \right)}\\ \end{align*}\end{split}\]

Note that \(\frac{P}{P_o} =P\) in pure number. Special case, \(P=1\)ATM, we denote \(\mu_o\) with \(\mu^{\ominus}\)\

Mixing of Ideal Gas#

\(G = n_A(\mu_A^\ominus + k_BT\ln P_A) + n_B(\mu_B^\ominus+k_BT\ln P_B)\)
letting \(x\) to be the mole fraction and using mole as a number unit
with the help of partial pressure(General chemistry) \(P_a=P \times x_A\)
\(\Delta G = n_\text{tot} RT(x_a\ln x_a + x_b \ln x_b)\)

Because mole fraction is always less than oe equal to 1, \(\Delta G\) is always negative for mixing process.
\(\Delta S = -\left(\frac{\partial \Delta G}{\partial T}\right) = -nR(x_a\ln x_a + x_b\ln x_b)\)
Similarly, entropy of mixing is always positive.

Chemical Potential for Liquid#

Pure liquid is usually marked with \(\ast\) .\

By mixing small amount of solute \(B\),

\[\mu_A(l) = \mu_A^\ominus + k_BT \ln P_A \cdots \text{\circled{2}}\]

\(\nwarrow\) no longer same as \(P_A^\ast\)
$\(\text{\circled{1} \& \circled{2}} \Rightarrow \mu_A^\ast + k_BT \ln\frac{P_A}{P_A^\ast} \cdots \circled{3}\)$

By the experiment by Raoult,

\[P_A = x_AP_A^\ast \cdots \text{Raoult's law}\]
\[\circled{3} \Rightarrow \mu_A(l) = \mu_A^\ast + k_BT \ln x_ a\]

\(\star\) Solute at low concentration (dilute soln)
\(\swarrow\) Pure solute vapor pressure

\(\times\)

Henry's law
$\rightarrow P_B = x_Bk_H$ constant\

Colligative Property#

$\star\( Boiling point elevation\ Derive \)\Delta T_b = K_{\text{BPE}} x_b$.\

\(\mu_A(g) = \mu_A(l) = \mu_A^\ast(l) + k_BT \ln x_A\)
\(\ln x_A = \frac{\mu_A(g) - \mu_A^\ast(l)}{k_BT} = \frac{\Delta G_\text{vap}}{k_BT}\)
\(\frac{d\ln x_A}{dT} = \frac{d}{dT}\left(\frac{\Delta G_\text{vap}}{k_BT}\right)\)
Recall
\(dG = -SdT + VdP \hspace{10pt} \left.\begin{array}[t]{l}\parfraci{G}{T}{P} = -S\\G = H - TS\end{array}\right\} \parfraci{G}{T}{P} = \frac{G-H}{T}\)
[HW]{.underline}
Derive \(\left(\frac{\partial}{\partial T}\left(\frac{G}{T}\right)\right)_P = -\frac{H}{T^2} \cdots \text{ Gibbs-Helmholtz Eq}\).

Using your result, derive

\[\frac{d}{dT}\ln x_A = \frac{1}{k_B} \frac{d}{dT}\left(\frac{\Delta G}{T}\right) = -\frac{1}{k_B} \frac{\Delta H_\text{vap}}{T^2}\]

$d\ln x_A = -\frac{\Delta H_\text{vap}}{k_B}\frac{1}{T^2}dT\(\ Integrate.\ \)\int_0^{\ln x_A} d\ln x_A = -\frac{\Delta H_\text{vap}}{k_B}\int_{T_b}^T\frac{1}{T^2}dT\(\ \)\ln x_A = -\frac{\Delta H_\text{vap}}{k_B}\left(\frac{1}{T_b}-\frac{1}{T}\right)$
\

$\begin{array}[t]{c} \ln (1- x_B)\\ \simeq -x_B\\ \because x_B \text{ small} \end{array}$ $\hspace{20pt} \frac{T-T_b}{TT_b} = \frac{\Delta T}{TT_b} \simeq \frac{\Delta T}{T^2_b}$

\(\uparrow\) Assume \(T \simeq T_b\)
So,

\[x_B = \underset{\text{Const. $K_{\text{BPE}}$}}{\underset{\downarrow}{\left(\frac{\Delta H_\text{vap}}{k_BT_b^2}\right)}}\Delta T //\]

\(\star\) Freezing Point Depression (Note: Solid is always pure.)
$\(\mu_A^\ast(s) = \mu_A(s) = \mu_A(l) = \mu_A^\ast(l) + k_BT\ln x_A\)$

[HW]{.underline}
Repeat the same procedure to obtain \(\Delta T_f = K_{\text{FPD}}x_B,\hspace{30pt} K_{\text{FPD}} = \frac{k_BT^2_f}{\Delta H_\text{fus}}\).
\(\star\) Solubility
At saturating point,
\(\mu_B^\ast(s) = \mu_B(l) = \mu_B^\ast(l) + k_BT\ln x_B\)
$\(\begin{array}{l} \frac{\mu_B^\ast(s) - \mu_B^\ast(l)}{k_BT} = \ln x_B\\ \text{\hspace{20pt}} \rotatebox{90}{=}\\ -\frac{\Delta G_\text{fus}}{k_BT}=\ln x_B\\ \frac{1}{k_B}\int_{T_1}^{T_2}\frac{\Delta H_\text{fus}}{T^2}dT = \int_0^{\ln x_B}d\ln x_B\\ \text{\hspace{30pt}} \Downarrow\\ \ln x_B = \frac{\Delta H_\text{fus}}{k_B}\left(\frac{1}{T_\text{f}}-\frac{1}{T}\right) \cdots \text{ideal solubility} \end{array}\)$
